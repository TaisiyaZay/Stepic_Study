С помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения. Git помогает вернуть файла в исходное состояние и видеть изменеия, внесённые в определённый период. Разработчик выполняет разные команды (например, commit, push), а все изменения синхронизируются с центарльным репозиторием. Git - это система контроля версий, а Github - онлайн-сервис, по сути социальная сеть. ОДна из основных целей Github - быть единым местом для поректов с исходным кодом. Предполагается, что пользователь делится чем-то полезным, а другеи люди смогут участвовать в разработке.
